PROGRAMAÇÃO EM PYTHON

Lafaiet Castro e Silva

O minicurso de introdução ao Python irá apresentar uma visão geral da linguagem, abordando:

  • Fundamentos
  • Estruturas de dados
  • Estruturas condicionais e de repetição 
  • Orientação a objetos
  • Principais módulos e frameworks

O curso terá um viés prático, onde os alunos terão oportunidade de colocar os conhecimentos adquiridos por meio de exercícios sugeridos. O objetivo é fornecer as bases para que os interessados possam então se apronfundar posteriormente.  De posse desses conhecimentos, e com dedicação pessoal, os alunos poderão fazer uso da linguagem e de seu ecosistema para trabalhar com desenvolvimento web, administração de sistemas e de infra estrutura, ciência de dados, etc. Todas áreas com alta demanda de profissionais bem qualificados com alta remuneração no mundo todo.
